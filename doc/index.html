Resumo:

Este documento apresenta os trabalhos desenvolvidos no escopo da disciplina Fundamentos de Computação Gráfica (FCG) - INF2608, cursada durante o Doutorado em Informática pela PUC-Rio . A disciplina de FCG é subdivida em três partes principais: Luz e Cor, Imagem e 3D. Para cada uma dessas partes é sugerido o desenvolvimento de um projeto. As seções a seguir detalham cada um desses projetos.



  • Luz e Cor

    O objetivo deste trabalho é melhor entender como se dá o processo aditivo de cor e analisar a linearidade deste processo. Para isso, foi implementado um programa que calcula as componentes CIEXYZ, CIExyY, CIELuv, CIELab e sRGB a partir do espectro, depois permite calcular estes mesmo componentes para o espectro resultante da soma de duas cores selecionadas.

  • Processamento de Imagem

    O objetivo é implementar alguns filtros de imagens. Em especial, nessa implementação focamos nos filtros geométricos: Perspective, Twirl e Sphere. Adicionalmente, também foi implementado o Filtro EWA para diminuir o aliasing.

  • Imagens 3D em Anaglifo

    O objetivo é estudar como se dá a criação de imagens 3D estereoscópicas pelo método anaglifo. Durante o estudo foram implementados alguns métodos de mapeamento de cor
